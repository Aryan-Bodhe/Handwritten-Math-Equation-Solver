1. Convolutional Neural Network
2. Number of Layers = 5
3. Saved at : https://iith-my.sharepoint.com/:f:/g/personal/ma23btech11005_iith_ac_in/EkbY9uZ8WiVIlp9Safw74TIBHDmZY5M232QHnupXCEe3sg?e=yqUleX
